# ForgeBGM 取扱説明書 / Instruction Manual (v3.0 GOLD)

## 1. 基本操作 / Basic Operations
1.  **インプット**: イメージを入力し、ターゲットモデル（Standard/HeartMuLa/ACE-Step）を選択します。
    **Input**: Enter your idea and select the target model (Standard/HeartMuLa/ACE-Step).
2.  **生成**: 「トータルプロデュース生成」をクリックすると、音楽構成、歌詞、ビジュアル案がタブごとに生成されます。
    **Generation**: Click "Generate Total Produce" to create music structure, lyrics, and art ideas in separate tabs.

## 2. ローカルAI音楽生成 / Local AI Generation
1.  **生成と再生**: 「ローカルで音楽生成＆再生」ボタンを押すと、あなたのPCのGPUを使用して音楽が生成されます。
    **Generate & Play**: Press "Generate & Play (Local)" to create music using your PC's GPU.
2.  **進捗管理**: ゲージが進み、完了すると自動的に再生が始まります。
    **Progress**: The progress bar shows the status. Music plays automatically upon completion.

## 3. タブ機能の活用 / Using Tabs
*   **🎵 音楽構成**: 外部AI（Suno等）向けのメインプロンプトとレポートを表示します。
    **Music**: Displays main prompts and reports for external AIs like Suno.
*   **✍ 歌詞案**: 曲の世界観に合わせたフレーズを提案します。
    **Lyrics**: Suggests phrases matching the world of the music.
*   **🎨 ビジュアル案**: ジャケット画像生成用のプロンプトを表示します。
    **Visual Art**: Displays prompts for jacket art generation.

## 4. ショートカットキー / Keyboard Shortcuts
*   **Ctrl + Enter**: プロンプト生成を実行します。
    **Ctrl + Enter**: Execute prompt generation.
*   **Ctrl + C**: プロンプトをコピーします。
    **Ctrl + C**: Copy prompt to clipboard.

## 5. 設定とトラブルシューティング / Troubleshooting
*   **GPUエラー**: グラフィックボードのドライバが最新であることを確認してください。
    **GPU Error**: Ensure your GPU drivers are up to date.
*   **起動しない**: 以前のバージョンで発生していた起動時エラーは、リソース管理の最適化により修正されました。
    **Startup Issue**: Startup errors in previous versions have been resolved via resource optimization.
