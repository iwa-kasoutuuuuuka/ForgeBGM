# ForgeBGM 取扱説明書 / Instruction Manual

## 1. 起動方法 / How to Start
`ForgeBGM.exe` を実行してください。ブラウザやサーバーの起動は不要です。
Run `ForgeBGM.exe`. No browser or server setup is required.

## 2. 基本操作 / Basic Operations
1.  **入力 / Input**: 「Input Prompt」欄に曲のイメージ（例：「雨の日のピアノ」「激しいボス戦」）を入力します。
    Enter the musical image (e.g., "rainy day piano", "intense boss battle") in the "Input Prompt" field.
2.  **生成 / Generate**: 「Generate」ボタンをクリックするか、ショートカットキーを押します。
    Click the "Generate" button or use the keyboard shortcut.
3.  **確認 / Review**: 発動したSKILL、構築されたプロンプト、詳細レポートが表示されます。
    Review the active SKILLs, constructed prompt, and detailed report.
4.  **コピー / Copy**: 各項目のコピーボタンをクリックして、外部の音楽生成AIに貼り付けてください。
    Click the copy button for each item and paste it into your external music generation AI.

## 3. ショートカットキー / Keyboard Shortcuts
*   **Ctrl + Enter**: プロンプト生成を実行 / Execute prompt generation
*   **Ctrl + C**: プロンプトをコピー（入力欄外で使用） / Copy prompt (when input field is not focused)

## 4. 強化プリセット / Enhanced Presets
画面上のボタンから、最適化済みの設定を即座に呼び出せます：
Instantly recall optimized settings using the on-screen buttons:
*   **Boss Battle**: 壮大な戦闘曲 / Epic battle music
*   **LoFi Chill**: 心地よい作業用BGM / Comfortable study BGM
*   **Cyberpunk**: 近未来的なエレクトロ / Futuristic electronic music

## 5. 生成レポートについて / About Generation Reports
「Generation Report」には、楽曲のBPM（テンポ）、Key（キー）、Seed値が含まれます。これらは音楽制作の参考資料として活用してください。
The "Generation Report" includes the BPM (tempo), Key, and Seed value. Use these as reference materials for your music production.
